
# MMM-Forecast-IO
Esta es una extensión de [MagicMirror] (https://github.com/MichMich/MagicMirror) que agrega un clima localizado usando la [API Dark Sky] (https://darksky.net/dev/) (originalmente Forecast.io) - - El mismo servicio que impulsa la [Aplicación Dark Sky] (https://darksky.net/app/).

Este módulo utiliza la [API de geolocalización] (https://developer.mozilla.org/en-US/docs/Web/API/Geolocation/Using_geolocation) para determinar la ubicación del espejo. Esto se puede desactivar configurando `latitud` y` longitud` en la configuración del módulo (ver [Opciones de configuración a continuación] (# opciones de configuración)).

A menos que se ejecute sin cabeza en un navegador, la geolocalización requiere la configuración de la [variable de entorno GOOGLE_API_KEY] (https://github.com/electron/electron/blob/master/docs/api/environment-variables.md#google_api_key).

! [captura de pantalla] (screenshot.png)

## Uso del módulo

Para usar este módulo, agréguelo a la matriz de módulos en el archivo `config / config.js`:
````javascript
modules: [
  {
    module: 'MMM-forecast-io',
    position: 'top_right',  // This can be any of the regions.
    config: {
      // See 'Configuration options' for more information.
      apiKey: 'abcde12345abcde12345abcde12345ab', // Dark Sky API key.
      // Only required if geolocation doesn't work:
      latitude:   16.77532,
      longitude: -3.008265
    }
  }
]
````
